# camera-app

Using:
  -react-native
  -react hooks
  -react-navigation
  -expo-cli
  -expo-media-library
  -expo-permissions
  -expo-camera

to create a simple app with:
  -a login page
  -access to a camera that:
    .accesses both cameras
    .has access to flash
    .and can save to your local files
  -access to your camera roll to:
    .preview images
